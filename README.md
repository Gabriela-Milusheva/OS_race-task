# OS_task-race
## Steps to get the project:
1. Clone the repository
 ```sh
git clone https://github.com/Gabriela-Milusheva/OS_race-task
 ```
 ```sh
cd OS_race-task
 ```
3. Create scripts and start the project:
Restart:
```sh
sh start.sh
```
Restart only race task:  
```sh 
sh race_start.sh
 ```
Restart only no race task:   
```sh 
sh no_race_start.sh
```
